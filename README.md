# 开发

## 安装依赖
```
    npm install 
```

## 启动命令

```
    // 运行一次即可
    npm run dll

    // 客户端渲染开发环境启动命令（客户端/服务端二选一）
    npm run start

    // 服务端渲染开发环境启动命令（客户端/服务端二选一）
    npm run start-ssr

    // 生产环境部署模式(基于PM2)
    npm run deply:server

    // mock模拟数据服务模式
    npm run mock:server
```
